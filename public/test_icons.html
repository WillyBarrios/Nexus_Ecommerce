Si ves estos iconos correctamente:

Entonces Bootstrap Icons está funcionando. El problema puede ser:

  • Cache del navegador (presiona Ctrl+F5 para refrescar)
  • Conexión a internet lenta
  • Bloqueador de contenido/ads

Si ves cuadrados negros:

Entonces hay un problema con la carga de Bootstrap Icons. Soluciones:

  • Verifica tu conexión a internet
  • Desactiva bloqueadores de contenido
  • Prueba en otro navegador
